Tasks App (Vue + TypeScript)
A small task manager built with Vue 3, TypeScript, and Vite.
It lets you add tasks, mark them as completed, remove them, and filter by all, active, or completed.
Features
- Add new tasks
- Mark tasks as done
- Remove tasks
- Filter tasks by status
